2. 下载并解压

具体操作可以查看 CentOS安装RocketMQ以及常用命令

在两台机器上都下载、解压好。
在rocketmq/conf目录下，有三种建议配置模式：

  • 2m-2s-async(2主2从异步) —— 本文采用这种
  • 2m-2s-sync (2主2从同步)
  • 2m-noslave (2主)


现在需要修改两台机器上2m-2s-async这个目录中的文件。
