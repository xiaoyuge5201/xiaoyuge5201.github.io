1.1 在末尾添加字段

1）语法：

语法格式的说明：

  • <表名> 为数据表的名字；
  • <字段名> 为所要添加的字段的名字；
  • <数据类型> 为所要添加的字段能存储数据的数据类型；
  • [约束条件] 是可选的，用来对添加的字段进行约束。
  • 这种语法格式默认在表的最后位置（最后一列的后面）添加新字段
